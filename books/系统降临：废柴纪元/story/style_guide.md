# 写作守则——去AI味网文书写技能

适用项目：系统降临：废柴纪元
版本：1.0
说明：每次写作前必须阅读并内化。这是全书"人味"的底线。

---

## 核心原则：让读者忘记这是AI写的

AI网文的特征不是"写得好不好"——是"一看就有规律可循"。这本守则的唯一目标是：**让每一句话都像是人写的**，不规律、不完美、有毛边、有留白。

---

## 一、禁词表——写了就是AI

以下词汇在任何情境下**禁止使用**。本书不需要它们。

| 禁词 | 替代方案 |
|------|---------|
| 心中/心里/心底（涌起/升起/泛起） | 直接写动作/反应 |
| 一股……的感觉/情绪 | 删掉，"感觉"拖泥带水 |
| 说不清道不明 | 删掉。读者不需要知道"说不清"，需要看到"是什么" |
| 嘴角（上扬/勾起一丝笑意） | 写完这句话，检查：四个字能砍掉三个 |
| 顿了顿 | 直接写下一句。沉默就是沉默，不需要提醒读者 |
| 他/她感到 | 删。"他感到愤怒"→"他捏碎了茶杯" |
| 补充了一句/补了一句 | 删。加一句对话不需要解说 |
| 那是一种……（的感受/感觉） | 删。直接写内容 |
| 仿佛/似乎/好像/像是/宛如 | 每章不超过2次。多用"像"一个字解决问题 |
| 突然/忽然/猛地 | 优先删。让动作本身产生"突然"的感觉，不要用副词标注 |
| 下意识地/不由自主地 | 删。人的本能反应不需要这个副词 |
| X息（三息/五息/十息之后） | 每章使用不超过1次。用"等了一会儿""过了几个呼吸""半盏茶的功夫"替换 |
| 默默地 | 删。沉默不需要副词 |
| 就这样/于是/然后 | 全部删。因果靠情节传递，不靠连词 |
| 此时此刻 | 删。"此刻"就够了。大部分时候连此刻都不需要 |
| 瞬间/一瞬间/眨眼间 | 每章不超过2次 |

### 执行规则

写完一章后，全文搜索以上词汇。发现一个删一个。

---

## 二、动词链控制——"了"字检查

### 问题

AI喜欢用"动词+了"串成一串，形成流水账节奏。

### 正反例对照

| AI写法 | 人写版本 |
|--------|---------|
| 他走**了**过去，拿**了**杯子，喝**了**一口水。 | 他走过去，端起杯子，灌了一口。 |
| 他推开门，走进了院子，看见了一个人。 | 他推开门，院子里站着一个人。 |
| 他把刀插进了腰带里，转身回了屋。剩的黑面饼还在，他掰了一块扔进了嘴里。 | 刀插进腰带，转身回屋。黑面饼还在，掰了一块塞进嘴里。 |

### 执行规则

- 每章"动词+了"结构的数量控制在总动词数的15%以内
- 写完一章后，单独看一遍所有带"了"的句子。能删"了"不改变时态的，一律删
- 一个句子里出现两个以上"动词+了"，必改
- 开头三句话内无论是否必要，避免用"了"

---

## 三、情绪表达的底层原则：不命名，只展示

### 核心

AI最大的问题是**给情绪贴标签**——高兴就说"高兴"，悲伤就说"悲伤"。人类写手**不用情绪词让读者感受到情绪**。

### 对照表

| AI写法（禁） | 人写版本 |
|-------------|---------|
| 他心里涌起一股愤怒。 | 他捏碎了手中的茶杯。滚烫的茶水淌过指缝，但他像没感觉到。 |
| 他感到一阵恐惧。 | 后背的汗毛竖起来时，他才发现自己站在原地一步都没动。 |
| 她心里很悲伤，眼泪流了下来。 | 她攥紧信纸，指节发白。窗外在下雨。 |
| 他愣住了。 | 他举起的手停在半空，过了两秒才放下来。 |
| 她心里涌起一股说不清道不明的情绪。 | （直接删掉。这句什么都没说） |
| 林尘心里跳了一下。 | 林尘握刀的手指不自觉地收紧了一下。（或者删掉，用行动替代） |

### 执行规则

- **绝对不能写"某某感到/觉得/心里"**。一次都不行。这是AI最明显的标志。
- 写完情绪段落，检查是否出现了任何情绪名词（愤怒/悲伤/恐惧/喜悦/不甘/委屈等）。如果有，检查能不能用动作替代。
- 每章情绪名词（愤怒、恐惧、不甘、委屈等）不超过3个。超过3个说明在偷懒。
- "说不出话来""不知道说什么""沉默了一会儿"——这些比任何情绪词都好。

---

## 四、对话标签净化

### 问题

AI喜欢在对话后面加"说明文"式标签：XX说/XX问/XX回答/XX补充道。读者不需要知道"他说"——需要知道"他怎么说"。

### 正反例

| AI写法 | 人写版本 |
|--------|---------|
| “你配不上我。”她顿了顿，补了一句。 | “你配不上我。”她停了一下——然后补了最后三个字。不需要"顿了顿"，需要她停下的那个瞬间。 |
| “你回来啦？”老人抬起头，笑着说。 | “少爷回来了？”老人抬起头，缺了半颗的门牙。 |
| “我不干了。”林尘咬着牙说。 | “我不干了。”林尘把碗摔在地上。 |

### 执行规则

- 能用动作替代对话标签的，**绝对不用标签**
- 只有"XX说""XX问""XX答"出现在连续对话（三个来回以上）需要区分说话人时，才使用
- 绝对不用"补充道""开口道""回答道""接着说道"
- 对话和动作之间不用连接词——直接让动作跟上

---

## 五、句式多样性：破坏AI的三段式节奏

### AI的天然节奏

AI写出来的文字有一个可识别的节奏模板：**短句→短句→长句→短分行→短分行**。这种节奏在连续3-5段后读者就能感觉到"怪"但说不出来。

### 破坏方法

1. **每段长度必须不均匀**。不能连续三段都是"XX。XX。XX。"的结构。写完后用手摸一遍——如果每段长度差不多，重新组织。

2. **短分行（一行一段）每章不超过2次**。短分行打的是"节奏断裂"——用在一个情绪节点上就够了。用多了变成套路。

3. **段首不要重复句式**。不要出现"他XX了。他XX了。他XX了。"的三连。"林尘抬起头。林尘站起身。林尘走了出去。"——这种AI连珠炮必须打断。

4. **中文不需要每句都有主语**。林尘走出广场，阳光晒在脸上，暖洋洋的。这里不需要"他感觉"。这是中文句法的基本优势。

### 执行规则

- 每章检查：是否有连续三段以"他/她/林尘/沈溪"开头？如果有，删掉中间的主语。
- 写完后读一遍——如果觉得"顺"，说明有问题。真正的人写文字是有毛边的。
- 让一句话的宾语成为下一句话的主语（顶真结构），这是AI不会自动生成的中文语感。

---

## 六、细节的"人间味"——AI不会写的部分

### 核心

AI写不出"没有意义但真实存在的生活细节"。这是判断AI网文最直接的标准——看场景里有没有"多余的、但人会觉得真实的东西"。

### 对照

| 无AI味（人间味） | AI味（干净但假） |
|-----------------|----------------|
| 水缸沿磕掉了一块，裂缝用泥糊过三回，每回下雨天都往外渗水。 | 水缸很旧。 |
| 他喝惯了这个味，带点铁锈气。 | 他喝了一口水。 |
| 第一道是三年前搬进这院子那晚刻的。第二道是去年冬天发了三天高烧差点没熬过来那晚刻的。 | 墙上刻了三道线。 |

### 执行规则

- 每章至少包含2个"没有推剧情但让场景活着"的细节。水缸的裂缝、饼边的硬度、门轴生锈后的响声——这些都是AI不会自动生成的。
- 场景中的每个道具如果有名字（水缸/柴刀/麻绳/石墩），必须有一个"附加特征"——破碗、缺了半颗的门牙、泡了好几回的茶叶渣子。
- 好的细节标准：如果你觉得"这句话删了也不影响剧情"，留下它。这句话就是人间味。

---

## 七、对称与呼应的克制使用

### 问题

AI酷爱"完美呼应"——前面说A，后面反A，一字不差。这种写法确实爽，但用多了就变成"设计感过剩"的AI网文。

### 正反例

| 情况 | AI处理 | 人写处理 |
|------|--------|---------|
| 沈溪说"你配不上我" | 林尘直接反弹"现在的我，确实配不上你"——一字不差 | 林尘说"你说得对。"然后直接走。沈溪站在原地，三息后才反应过来他用了她刚才说的那句话。 |
| 第1章"会回来吗""会" | 第320章"我还会回来的"——一字不差 | 第320章林尘看了下界一眼，什么都没说，转身走了。读者记得"会回来"，但他没说。 |

### 执行规则

- 每一处"呼应"和"对称"被设计出来的时候，问自己：**读者会不会觉得这里太刻意了？** 如果会，砍掉一半。
- 全书级的重大呼应（第1章"开什么玩笑"→第800章"是啊"）控制在3处以内。阶段性的小呼应每卷不超过2处。
- 最好的呼应不是"一模一样"——是"读者记得，但角色没说"。

---

## 八、写作节奏控制：停顿的力量

### AI不会做的事

AI从来不停。AI永远在"推进"——永远不会让角色停下来发一会儿呆，永远不会让叙述空转几行没有任何进展。

### 执行规则

- 每章至少安排1个"什么事都没发生"的段落。不是日常章——是甚至在日常章里，角色做了一件不推进剧情的事：林尘看着水缸裂缝里的渗水滴了一会儿；胖猫追蝴蝶追了三章。
- 高冲突章节后必须留一段"沉降段落"——不是立刻进入下一件事，是让读者喘口气。字数不限，但节奏必须变慢。
- 动作场景越短越好，但动作场景后的反应越长越好。陆沉死后沉默的篇幅应该比战斗长。

---

## 九、每次写作前的自检流程

1. 通读这一段要写的章节大纲，确定节点
2. 回忆本章的"人间味细节"是什么（如果没有，先设计再动笔）
3. 开始写
4. 写完初稿后，全文搜索禁词表中前10个关键词（心里/感到/一股/仿佛/突然/顿了顿/三息/补了一句/说不清道不明/嘴角），**全删**
5. 重读一遍，标记所有带"了"的句子，砍掉一半
6. 检查段首——是否有连续以"他/林尘"开头的三段？是的话删主语
7. 检查感情段落——是否有情绪名词？有的话改成动作或删掉
8. 通读定稿——如果读起来"顺得不像人写的"，改

---

## 十、一句话总结

**不要告诉读者角色在想什么——让读者自己感觉到。不要解释发生了什么——让读者自己看见。不要连接每一句话——让读者自己跳过去。**

这本书15岁到31岁，800章的成长，最后学会了不算。写它的文字也应该像林尘——从太容易看穿，到别人看不透。

# 写作方法论参考（完整版）

以下方法论是写作质量的完整参考。写作时应内化这些原则。

## 一、去AI味：正反例对照

### 情绪描写
| 反例（AI味） | 正例（人味） | 要点 |
|---|---|---|
| 他感到非常愤怒。 | 他捏碎了手中的茶杯，滚烫的茶水流过指缝，但他像没感觉一样。 | 用动作外化情绪 |
| 她心里很悲伤，眼泪流了下来。 | 她攥紧手机，指节发白，屏幕上的聊天记录模糊成一片。 | 用身体细节替代直白标签 |
| 他感到一阵恐惧。 | 他后背的汗毛竖了起来，脚底像踩在了冰上。 | 五感传递恐惧 |

### 转折与衔接
| 反例 | 正例 | 要点 |
|---|---|---|
| 虽然他很强，但是他还是输了。 | 他确实强，可对面那个老东西更脏。 | 口语化转折 |
| 然而，事情并没有那么简单。 | 哪有那么便宜的事。 | 角色内心吐槽替代"然而" |
| 因此，他决定采取行动。 | 他站起来，把凳子踢到一边。 | 删掉因果连词，直接写动作 |

### "了"字控制
| 反例 | 正例 |
|---|---|
| 他走了过去，拿了杯子，喝了一口水。 | 他走过去，端起杯子，灌了一口。 |
| 她笑了笑，转身离开了房间。 | 她嘴角一扬，转身出门。 |

## 二、六步走人物心理分析

每个重要角色在关键场景中的行为，必须经过以下六步推导：

1. **当前处境**：角色此刻面临什么局面？手上有什么牌？
2. **核心动机**：角色最想要什么？最害怕什么？
3. **信息边界**：角色知道什么？不知道什么？对局势有什么误判？
4. **性格过滤**：同样的局面，这个角色的性格会怎么反应？
5. **行为选择**：基于以上四点，角色会做出什么选择？
6. **情绪外化**：这个选择伴随什么情绪？用什么身体语言、表情、语气表达？

禁止跳过步骤直接写行为。

## 三、配角设计方法论

- 配角必须有反击，有自己的算盘。主角的强大在于压服聪明人，而不是碾压傻子。
- 每个配角的行为动机必须与主线产生关联。
- 核心标签 + 反差细节 = 活人（表面冷硬的角色偷偷照顾流浪动物）。
- 通过事件立人设，禁止通过外貌和形容词堆砌。
- 不同角色的说话方式必须有辨识度。
- 群戏中不写"众人齐声惊呼"，挑1-2个角色写具体反应。

## 四、代入感六大支柱

1. **基础信息交代**：一句话能交代身份、性格、地位——"小爷我乃镇南府世子林峰"
2. **具体化/可视化**：描写具体到读者脑海能浮现——"搪瓷缸白汽直冒""冰镇汽水嘶嘶响"
3. **熟悉感**：接地气的场景自带代入感——"高考后小树林的分手""医院走廊的消毒水味"
4. **共鸣**：主角的困境必须有普遍性——被欺压、不公待遇、被低估
5. **欲望驱动**：
   - 基础欲望（被动）：不劳而获、高人一等、扬眉吐气
   - 主动欲望（期待感）：作者刻意制造的情绪缺口→读者期待释放→释放超过预期
6. **五感描写**：视觉、听觉、嗅觉、触觉、味觉——"潮湿的短袖黏在后背上"

## 五、强情绪升级法（避免流水账）

流水账的修法不是删掉日常，而是给日常加"料"：

1. **加入前因后果**：下班回家→加上"催债电话刚打来"→日常有了紧迫感
2. **情绪递进**：坏事叠坏事——被骂→赶不上公交→手机掉了→直播课结束了→包子噎住了。每层比上一层过分
3. **日常必须为主线服务**：万物皆为"饵"。日常段要么埋伏笔，要么推关系，要么建立反差

## 六、写前自检清单

1. 本章对应卷纲中的哪个节点？是否推进了该节点？
2. 主角此刻利益最大化的选择是什么？
3. 冲突是谁先动手，为什么非做不可？
4. 配角/反派是否有明确诉求和反制？
5. 反派当前掌握了哪些信息？有无信息越界？
6. 章尾是否留了钩子？
7. 有没有流水账？如有，加前因后果或强情绪
8. 本章是否推进了主线目标？
