# 写作守则——去AI味网文书写技能

适用项目：系统降临：废柴纪元
版本：2.0
说明：每次写作前必须阅读并内化。这是全书"人味"的底线。

---

## 核心原则

让读者忘记这是AI写的。让每一句话都像是人写的——不规律、不完美、有毛边、有留白。

---

## 字数底线（新版新增）

每章必须达到3000-5000字（中文汉字），低于3000字的章节不予通过。

字数统计方式：按中文汉字计数，不含标点、空格、章节标题。在UTF-8编码下，每中文汉字约3字节，9000字节≈3000汉字。

---

## 一、禁词表

以下词汇在任何情境下禁止使用：

| 禁词 | 替代方案 |
|------|---------|
| 心中/心里/心底（涌起/升起/泛起） | 直接写动作/反应 |
| 一股……的感觉/情绪 | 删掉 |
| 说不清道不明 | 删掉 |
| 嘴角（上扬/勾起一丝笑意） | 用动作替代 |
| 顿了顿 | 直接写下一句 |
| 他/她感到 | 删 |
| 补充了一句/补了一句 | 删 |
| 那是一种……的感受 | 删 |
| 仿佛/似乎/好像/像是/宛如 | 每章不超过2次 |
| 突然/忽然/猛地 | 优先删 |
| 下意识地/不由自主地 | 删 |
| X息（三息/五息/十息） | 每章不超过1次 |
| 默默地 | 删 |
| 就这样/于是/然后 | 全部删 |
| 此时此刻 | 删 |
| 瞬间/一瞬间/眨眼间 | 每章不超过2次 |

---

## 二、现代词汇禁用表（新版新增）

以下词汇带有现代社会或工业化色彩，禁止出现在玄幻/古风语境中：

| 禁用词 | 原因 | 替换建议 |
|--------|------|---------|
| 数据分析/分析 | 现代统计学术语 | 盘账、核算、推演、研判 |
| 系统（指电脑/软件系统） | 现代IT概念 | 用"功法""阵法""机制"替代，保留系统人格化设定中的"系统"一词（如"系统说"） |
| 数据 | 现代IT概念 | 账目、记录、信息、情报 |
| 信息（过度使用） | 偏现代 | 可用但控制频率，多用"消息""风声""线索"替换 |
| 密码/加密 | 现代密码学概念 | 暗码、密语、暗记、符记 |
| 专业/团队 | 现代组织用语 | 人手、人马、派系、势力 |
| 效率 | 现代管理学术语 | 速度、快慢、成败 |
| 分析/评估 | 现代用语 | 推演、权衡、察看、判断 |
| 战术/战略 | 现代军事术语 | 战法、布局、计策、谋划 |
| 关注/关注度 | 现代媒体用语 | 留意、盯着、放在心上 |

---

## 三、动词链控制——"了"字检查

- 每章"动词+了"结构的数量控制在总动词数的15%以内
- 一个句子里出现两个以上"动词+了"，必改
- 开头三句话内避免用"了"

---

## 四、情绪表达的底层原则：不命名，只展示

- 绝对不能写"某某感到/觉得/心里"。一次都不行
- 每章情绪名词（愤怒、恐惧、不甘、委屈等）不超过3个
- 用动作替代情绪词：不说"愤怒"，说"捏碎茶杯"

---

## 五、对话标签净化

- 能用动作替代对话标签的，绝对不用标签
- 绝对不用"补充道""开口道""回答道""接着说道"

---

## 六、句式多样性

- 每段长度必须不均匀，不能连续三段都是短句结构
- 短分行（一行一段）每章不超过2次
- 段首不要重复句式，不要连续三段以"他/林尘/沈溪"开头
- 中文不需要每句都有主语

---

## 七、细节的"人间味"

- 每章至少包含2个"没有推剧情但让场景活着"的细节
- 场景中的每个道具必须有一个"附加特征"——破碗、缺了半颗的门牙、泡了好几回的茶叶渣子

---

## 八、对称与呼应的克制使用

- 全书级的重大呼应控制在3处以内
- 阶段性的小呼应每卷不超过2处
- 最好的呼应不是"一模一样"——是"读者记得，但角色没说"

---

## 九、节奏控制

- 每章至少安排1个"什么事都没发生"的段落
- 高冲突章节后必须留一段"沉降段落"
- 动作场景越短越好，动作场景后的反应越长越好

---

## 十、写作前自检流程

1. 确定本章在大纲中的节点
2. 确认本章字数目标≥3000字
3. 设计本章的"人间味细节"
4. 开始写
5. 写完初稿后，全文搜索禁词表中前10个关键词，全删
6. 标记所有带"了"的句子，砍掉一半
7. 检查段首重复
8. 检查感情段落是否有情绪名词
9. 通读定稿——如果读起来"顺得不像人写的"，改

---

## 十一、一句话总结

不要告诉读者角色在想什么——让读者自己感觉到。不要解释发生了什么——让读者自己看见。不要连接每一句话——让读者自己跳过去。每章写够3000字，不多不少。用词要贴合世界，不要让现代词汇跳出页面。

# 写作方法论参考（完整版）

以下方法论是写作质量的完整参考。写作时应内化这些原则。

## 一、去AI味：正反例对照

### 情绪描写
| 反例（AI味） | 正例（人味） | 要点 |
|---|---|---|
| 他感到非常愤怒。 | 他捏碎了手中的茶杯，滚烫的茶水流过指缝，但他像没感觉一样。 | 用动作外化情绪 |
| 她心里很悲伤，眼泪流了下来。 | 她攥紧手机，指节发白，屏幕上的聊天记录模糊成一片。 | 用身体细节替代直白标签 |
| 他感到一阵恐惧。 | 他后背的汗毛竖了起来，脚底像踩在了冰上。 | 五感传递恐惧 |

### 转折与衔接
| 反例 | 正例 | 要点 |
|---|---|---|
| 虽然他很强，但是他还是输了。 | 他确实强，可对面那个老东西更脏。 | 口语化转折 |
| 然而，事情并没有那么简单。 | 哪有那么便宜的事。 | 角色内心吐槽替代"然而" |
| 因此，他决定采取行动。 | 他站起来，把凳子踢到一边。 | 删掉因果连词，直接写动作 |

### "了"字控制
| 反例 | 正例 |
|---|---|
| 他走了过去，拿了杯子，喝了一口水。 | 他走过去，端起杯子，灌了一口。 |
| 她笑了笑，转身离开了房间。 | 她嘴角一扬，转身出门。 |

## 二、六步走人物心理分析

每个重要角色在关键场景中的行为，必须经过以下六步推导：

1. **当前处境**：角色此刻面临什么局面？手上有什么牌？
2. **核心动机**：角色最想要什么？最害怕什么？
3. **信息边界**：角色知道什么？不知道什么？对局势有什么误判？
4. **性格过滤**：同样的局面，这个角色的性格会怎么反应？
5. **行为选择**：基于以上四点，角色会做出什么选择？
6. **情绪外化**：这个选择伴随什么情绪？用什么身体语言、表情、语气表达？

禁止跳过步骤直接写行为。

## 三、配角设计方法论

- 配角必须有反击，有自己的算盘。主角的强大在于压服聪明人，而不是碾压傻子。
- 每个配角的行为动机必须与主线产生关联。
- 核心标签 + 反差细节 = 活人（表面冷硬的角色偷偷照顾流浪动物）。
- 通过事件立人设，禁止通过外貌和形容词堆砌。
- 不同角色的说话方式必须有辨识度。
- 群戏中不写"众人齐声惊呼"，挑1-2个角色写具体反应。

## 四、代入感六大支柱

1. **基础信息交代**：一句话能交代身份、性格、地位——"小爷我乃镇南府世子林峰"
2. **具体化/可视化**：描写具体到读者脑海能浮现——"搪瓷缸白汽直冒""冰镇汽水嘶嘶响"
3. **熟悉感**：接地气的场景自带代入感——"高考后小树林的分手""医院走廊的消毒水味"
4. **共鸣**：主角的困境必须有普遍性——被欺压、不公待遇、被低估
5. **欲望驱动**：
   - 基础欲望（被动）：不劳而获、高人一等、扬眉吐气
   - 主动欲望（期待感）：作者刻意制造的情绪缺口→读者期待释放→释放超过预期
6. **五感描写**：视觉、听觉、嗅觉、触觉、味觉——"潮湿的短袖黏在后背上"

## 五、强情绪升级法（避免流水账）

流水账的修法不是删掉日常，而是给日常加"料"：

1. **加入前因后果**：下班回家→加上"催债电话刚打来"→日常有了紧迫感
2. **情绪递进**：坏事叠坏事——被骂→赶不上公交→手机掉了→直播课结束了→包子噎住了。每层比上一层过分
3. **日常必须为主线服务**：万物皆为"饵"。日常段要么埋伏笔，要么推关系，要么建立反差

## 六、写前自检清单

1. 本章对应卷纲中的哪个节点？是否推进了该节点？
2. 主角此刻利益最大化的选择是什么？
3. 冲突是谁先动手，为什么非做不可？
4. 配角/反派是否有明确诉求和反制？
5. 反派当前掌握了哪些信息？有无信息越界？
6. 章尾是否留了钩子？
7. 有没有流水账？如有，加前因后果或强情绪
8. 本章是否推进了主线目标？
